XSCeon 2019 Updates

Season 9 Releases!

With a map size of over 1GB and around 200 hours of gametime, the XS team is proud to announce Season 9 of XS-19.

What's new? Season 9 presents a cleaner map than before, fixed major bugs, and reinstatement of many new mods. Season 9 does NOT feature a reset of the End, but plans for a complete overhaul will be present in Season 10.

Current Server State

As of right now, XS-2019 is highly optimized and more bug free than Season 8's release. We're hoping to move to a higher end server host in a couple months for more flexibility.

  • Approximatly 210 Overworld chunks have been generated with 9 total chunks being deleted (5% Corruption Rate)
  • Around 70 End chunks have been generated with ~30 chunks corrupted (~40% corruption rate). As of right now, the End will no longer be optimized with a possible reset next season.
  • Around 22 Nether chunks have been generated, none have ever been corrupted (0% Corruption rate)

*Corruption rates are based on the entirety of Season 8 (~6 months)

Season 8 Changelog

  • 7/15/23: Season 7.0 Releases - Server optimizations, view count increase to 8 (+2), minor anticheats installed, updated to Minecraft Paper Version 1.20.1
  • 11/8/23: Season 8.0 Release - Server optimizations, bug fixed for tick timing. Server version still 1.20.1

Crate Drops!

Season 8 has less updates to the server as a whole, but new armor crate sets will be dropping! Our current drops:

  • Kronos Set - Featuring Protection 5 Armor, and a powerful set of tools
  • Tartarus Set - Protection 10 Armor with the rarest trims, even more godly set of tools, and the life/death potions.
  • NEW: Skylivier Set - Rare blue trimmed Protection 4 Armor with feather falling 8 boots and a Punch 3 Bow with Power 6 and Flame 3.
  • NEW: Arthier Set - Rare orange trimmed armor with Flame Protection 5 and Blast resistance 5. An axe includes Fire Aspect 5, Sharpness 5, and efficiency 5. A pickaxe includes sharpness 5 and efficiency 8.

The Kronos Set and Tartarus Set have been both found by faction Vexian. The Tartarus set is currently the most advanced set to release with maximum protection meant to destroy enemies. The new Skylivier set (sky-live-ier) is meant to be the ultimate set for taking control of the skies. The Arthier set (arth-i-er) is the ultimate nether domination set.
